## Fastest Way to Get Started

### Option A: No Git/Command Line Needed
1. Go to https://github.com/new
2. Name: `vision-quest`
3. Click "Create repository"
4. Click "uploading an existing file"
5. Drag all files from this ZIP
6. Commit changes
7. Done! ✅

### Option B: Using Git (If Installed)
```bash
# Navigate to project folder
cd vision-quest-project

# Initialize and push
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/vision-quest.git
git branch -M main
git push -u origin main
```

---

## Enable Live Website

1. Go to repository Settings
2. Click "Pages" (left sidebar)
3. Source: Deploy from branch
4. Branch: main → root
5. Save
6. Wait 1-2 minutes
7. Visit: `https://YOUR-USERNAME.github.io/vision-quest/`
